HHKB Professional 2键盘真的适合你吗？

发布 : 2018-06-24 分类 : 评测 浏览 :

我使用HHKB键盘到现在已有近2年的时间了。在这些日子，HHKB 给我的工作带来了很大便捷。但是，此时的你如果也想购买HHKB，我反而会告诉你 HHKB 也许不适合你……

为什么不适合你？让我们先简单地了解下 HHKB

首先 HHKB 是静电电容键盘

关于静电电容键盘，百度百科是这样说的：“静电容键盘是利用电容容量的变化来判断按键的开和关，在按下按键后，开关中电容容量发生改变，从而实现触发，整个过程不需要开关的闭合。” 其实作为使用者来说我只需知道，静电电容键盘和机械键盘相比：按键声音相对小、使用寿命较长还有就是相对贵些。

还有就是 HHKB 60% 的按键布局

HHKB静电电容键盘的布局和我们常见的键盘布局有所不同，这些不同是由东京大学名誉教授和田英一的所设计的，这些不同是为了减少用户在正常输入状态下手掌移动距离。 HHKB 的布局最明显的地方就是 F1 ~ F12 和方向按键的使用需要配合 FN 按键使用。 并且HHKB 将 “~” 按键替换为 “ESC” 按键，而 “~” 按键放到了 “回退” 按键的位置。并且 “回退” 按键向下移动了一个位置，也就是 “|” 按键的位置。还有就是HHKB 去除了大写锁定按键。并且替换为了 “Ctrl” 按键。

为什么不适合你？让我告诉你为什么我会选择 HHKB

其实我选择 HHKB 最重要的原因就是因为它的布局更加的适配MacOS。（当然 HHKB 的DIP开关调节可以设置适配 Windows 系统。） 个人感觉 HHKB 主要面向的是专业性领域，比如 程序员、Linux运维人员等等。而对于我，一个 Vim 党，HHKB 可以说是神器了。而对于使用集成开发环境的小伙伴也许就不那么适合了，应为好多快捷键使用起来不是很方便。

为什么不适合你？那让我告诉你 HHKB 给我带来了什么？

让我不再想什么时候换下一个键盘，在 HHKB 之前我也使用过机械键盘，当然并不多。而拥有了HHKB，我的想法就是，我要一直用下去。即使有一天我不做程序员了，我也会把它放到一个重要的位置。 当然 HHKB 除了能让我小小的装一下以外，它给我的工作带来了很大的帮助。前面说过，HHKB 为了减少使用过程中手掌的移动距离，做了很大的改变。而这些改变对于我这样的一个Vim 党来说，简直可以说犹如神助。 首先要说的就是 Ctrl 键取代低使用频率大写锁定按键，这一点可以让我的小拇指来点击 Ctrl 按键，这一点的改变真是谁用谁知道啊！ 还有就是 Esc 按键和 Delete 按键位置设计，同样这两个按键可以让我的小拇指在手掌位置不动的情况下轻松的按到。 HHKB键盘上这些看似很小的变动，极大的提升了我的工作效率。

为什么不适合你？我如何克服 HHKB 的不利之处

当然 HHKB 还是有些不大方便的地方，比如说方向按键，而我大多数情况会选择 Ctrl + p、n、b、a、e、f，来操作光标的移动。还有就是 F1~ F12 按键，还好的是我的频率并不高，可以忽略。

那么 HHKB 到底是不适合你？

于是乎啰嗦了这么多，我就是想告诉你：如果你习惯使用 Vim 或者 Emacs 来敲代码，又或者说常常使用命令行来进行工作，那么 HHKB 或许是你不二的选择。而如果你习惯使用集成开发环境，甚至只是想买来玩玩游戏，那么也许其它的机械键盘更加的适合你。
